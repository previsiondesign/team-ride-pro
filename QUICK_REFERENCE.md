# Quick Reference: Visual Editing → Cursor

## Chrome DevTools → Cursor (Most Common)

### 1. Make Visual Change
- `F12` → Click element → Edit CSS in Styles panel
- Changes appear instantly

### 2. Copy CSS Back
- Right-click CSS rule → "Copy rule"
- Or manually copy the property: `padding: 8px;`

### 3. Paste in Cursor
- `Ctrl+F` → Search for class name (e.g., `.mobile-group-card`)
- Find the CSS rule
- Replace the property value
- Save (`Ctrl+S`)

### 4. Test
- Refresh browser (`F5`)
- Verify change is saved

---

## Figma → Cursor

1. **Design in Figma** → Get CSS values
2. **In Cursor:** Find element → Add/update CSS
3. **Test in browser** → Fine-tune in DevTools
4. **Copy back to Cursor**

---

## Webflow → Cursor

1. **Design in Webflow** → Export code
2. **In Cursor:** Replace HTML/CSS sections
3. **Keep your JavaScript intact!**
4. **Test functionality** → Fix in DevTools if needed

---

## Pro Tips

- **Always copy DevTools changes immediately** - they're lost on refresh
- **Use Cursor search** (`Ctrl+F`) to find CSS classes quickly
- **Test one change at a time** - easier to track what works
- **Use comments** in CSS to mark what you changed
